# 快速入门指南

## 🚀 5分钟快速设置

### 步骤 1: 安装必需软件

1. **Obsidian**: [下载地址](https://obsidian.md/)
2. **VS Code**: [下载地址](https://code.visualstudio.com/)
3. **Git**: [下载地址](https://git-scm.com/)

### 步骤 2: 克隆或创建仓库

```bash
# 方法一: 克隆现有仓库
git clone https://github.com/your-username/knowledge-base.git
cd knowledge-base

# 方法二: 创建新仓库
mkdir my-knowledge-base
cd my-knowledge-base
git init
# 复制本项目的配置文件到你的目录
```

### 步骤 3: 配置 Obsidian

1. 打开 Obsidian
2. 选择 "Open folder as vault"
3. 选择你的知识库目录
4. Obsidian 会自动读取 `.obsidian/` 目录中的配置

### 步骤 4: 配置 VS Code

```bash
# 在 VS Code 中打开工作区
code knowledge-base.code-workspace
```

VS Code 会提示安装推荐的扩展，点击安装即可。

### 步骤 5: 第一次使用

```bash
# 创建今日笔记
./scripts/new-daily-note.sh

# 生成知识库索引
./scripts/generate-index.sh
```

## 📝 基本使用流程

### 日常笔记工作流

1. **开始一天的工作**:
   ```bash
   ./scripts/new-daily-note.sh
   ```

2. **在 Obsidian 或 VS Code 中编写笔记**
   - 使用快捷键创建模板
   - 利用 GitHub Copilot 辅助写作

3. **结束工作时同步**:
   ```bash
   ./scripts/sync-notes.sh
   ```

### 创建不同类型的笔记

#### 在 Obsidian 中:
- 按 `Ctrl+N` 创建新笔记
- 按 `Ctrl+T` 插入模板

#### 在 VS Code 中:
**基础模板**:
- 输入 `daily` + Tab: 创建日记模板
- 输入 `project` + Tab: 创建项目模板
- 输入 `meeting` + Tab: 创建会议模板
- 输入 `learning` + Tab: 创建学习笔记模板

**知识管理方法论模板**:
- 输入 `evergreen` + Tab: 创建常青笔记模板
- 输入 `moc` + Tab: 创建内容地图(MOC)模板
- 输入 `zettel` + Tab: 创建卢曼卡片盒模板

> 💡 **新功能**: 本系统现已支持三种主流知识管理方法论！详见 [知识管理方法论指南](知识管理方法论指南.md)

## 🛠️ 常见问题解决

### Q: Obsidian 插件无法加载？
A: 确保插件在 `.obsidian/community-plugins.json` 中列出，并且已启用。

### Q: VS Code 中 Copilot 无法使用？
A: 检查是否已登录 GitHub 账号并订阅了 Copilot 服务。

### Q: Git 同步冲突？
A: 使用以下命令解决:
```bash
git status
git diff
# 手动编辑冲突文件
git add .
git commit -m "resolve conflicts"
```

### Q: 如何在移动设备上使用？
A: 
1. 安装 Obsidian Mobile
2. 使用 Working Copy (iOS) 或 Termux (Android) 进行 Git 同步
3. 或使用 Obsidian Sync / Git 插件

## 📱 移动端配置

### iOS 设置
1. 安装 Obsidian Mobile
2. 安装 Working Copy
3. 在 Working Copy 中克隆仓库
4. 在 Obsidian 中打开 Working Copy 文件夹

### Android 设置
1. 安装 Obsidian Mobile  
2. 安装 Termux
3. 在 Termux 中:
   ```bash
   pkg install git
   git clone <your-repo>
   ```
4. 在 Obsidian 中打开文件夹

## 🎨 自定义配置

### 修改主题
1. 将主题文件放在 `.obsidian/themes/` 目录
2. 在 Obsidian 设置中选择主题

### 添加自定义 CSS
1. 在 `.obsidian/snippets/` 目录添加 `.css` 文件
2. 在 Obsidian 设置中启用代码片段

### 自定义快捷键
编辑 `.obsidian/hotkeys.json` 文件添加自定义快捷键。
