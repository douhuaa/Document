## 📝 使用说明

### 基本使用流程
1. **📋 选择提示词**: 根据开发需求浏览分类目录
2. **📝 复制模板**: 复制对应的提示词模板
3. **🔧 定制内容**: 根据具体项目需求调整参数
4. **🤖 生成代码**: 在 GitHub Copilot 中使用提示词
5. **✅ 验证结果**: 测试生成的代码并优化

### 提示词使用技巧
- **具体化描述**: 提供详细的功能需求和约束条件
- **包含示例**: 在提示词中包含输入输出示例
- **分步说明**: 对复杂逻辑分步骤描述
- **指定格式**: 明确代码风格和注释要求
- **测试验证**: 始终验证生成代码的正确性
